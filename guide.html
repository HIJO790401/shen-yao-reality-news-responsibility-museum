閱讀導覽 Guide

如何閱讀平台、新聞台、博物館與十四軸分工。How to read the platform, reports, museum, and the 14-axis newsroom.

平台怎麼閱讀 / How to read this platform

先由首頁四大入口理解系統：新聞台、博物館、十四軸、官方入口。
Start from the four core gateways on Home: reports, museum, anchors, and official portal.

新聞台怎麼看 / How to read reports

以日期、分類與責任鏈為索引，先看摘要再進入詳情。
Use date, category, and responsibility chain as reading indexes.

責任博物館怎麼看 / How to read museum

先辨識館別，再進入館藏編號與導覽說明。
Identify hall first, then read each collection ID and guide notes.

十四軸分工 / 14-axis division

十四軸不是娛樂設定，而是編輯部的責任工位。
The 14 axes are responsibility roles, not fan-character settings.

創辦人與族人關係 / Founder-team relation

創辦人負最終生命責任與最終發布，族人依職能對齊。
Founder holds final release and life responsibility; team aligns by function.

建議閱讀順序 / Suggested order

Guide → Reports → Museum → Anchors → Founder。
Follow the sequence to establish full context before judging high-risk content.
